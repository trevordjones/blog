Learning to Code

In my first year of being a software developer I found out that it can be extremely hard to learn when you're just starting out.

This blog will be helping those who are just learning to code (and those who want to learn more!). I can write from the point of view of someone who learned to code without a technical background and can go over things I didn't understand.

I love all things programming/web/the Internet of Things. It's an exciting time to be a developer!

Feel free to follow me on Github to check out my latest projects.

Read through my posts below and enjoy!
